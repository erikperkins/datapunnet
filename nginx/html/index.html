Welcome to the Ecosystem

MNIST Digit Classification

This website is intended to be an exhibition of different web development and data science techniques. Architecturally, it consists of several applications implemented in different languages and frameworks, organized in a service-oriented fashion. External APIs are employed as well, to collect data and distribute computations. The user interface is implemented via a collection of JavaScript and CSS libraries.

In greater detail, the composition of this site is as follows. The application infrastructure consists of

  • Amazon Web Services EC2 for hosting
  • CircleCI for continuous integration
  • Github for version control
  • PostgreSQL for structured data storage
  • Amazon Web Services S3 for unstructured data storage
  • Amazon Web Services EMR for distributed computing

The application implementation has the following components

  • Ruby on Rails for the main application functionality
  • Python/Django for machine learning processes
  • Haskell/Snap for user authentication
  • Scala/Scalatra for distributed computing with Spark
  • AngularJS for the user interface
  • D3JS for data visualization
  • Bootstrap for styling

The three subordinate applications expose RESTful JSON APIs, through which the main Rails application communicates with them via ActiveResource. In the main application, some functionality is extracted to a Rails engine. The Scala/Scalatra application uses categorical design patterns (monoids, applicative functors, monads, algebraic data types, etc.) wherever possible, while the Haskell/Snap application uses these patterns essentially by default.
